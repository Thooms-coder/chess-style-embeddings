# Abstract

This project studies whether chess player weaknesses can be modeled in a way that is **invariant to rating**. Instead of treating mistakes as isolated blunders, the system estimates expected move quality for a given rating and then analyzes residual deviations from that expectation. Using publicly available rated Lichess Standard games, filtered to Rapid and Classical players between 1200 and 2200 rating, the pipeline parses PGN archives, hashes player identifiers, annotates games with Stockfish centipawn-loss labels, and converts games into fixed 128-ply training windows with structured move features.

The modeling pipeline uses a transformer encoder to produce player-conditioned latent representations from move sequences. Initial experiments showed that raw move-level residual prediction was too noisy, so the residual task was reformulated as **phase-aggregated weakness prediction** across opening, middlegame, and endgame. This phase-level target became the first residual formulation that learned useful signal consistently as dataset size increased.

On the current best run with 25,000 Stockfish-annotated games, the model achieved a validation phase-residual RMSE of 45.76, improving over earlier 5k and 10k experiments. Evaluation also shows that embedding-based nearest-neighbor matching captures player weakness profiles better than rating-only matching, with lower profile RMSE and a 57.38% win rate over rating-based neighbors. Temporal evaluation shows that player weakness profiles are more stable across chronological splits than random baselines, and short-horizon personalization tests show modest but positive alignment between earlier predicted weaknesses and later observed weaknesses.

Overall, the project provides evidence that rating-aware residual modeling and latent player embeddings can recover player-specific structure beyond raw skill, while also showing that personalized forecasting remains the weakest part of the system and requires more scale for stronger claims.
